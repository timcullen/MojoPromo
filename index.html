    MoneyMojo uses the very latest cloud based technology, so your data is available wherever you are, and whatever device you are using. All you need is internet access. Windows? Mac? Tablet? Mobile phone? MoneyMojo works with all of them. No need to download the latest version of the software. Whenever we issue new updates (which we do all the time) you have access to the new features immediately and automatically. likewise backups of all your data are undertaken centrally so you can relax in the knowledge your data is safe and secure.
